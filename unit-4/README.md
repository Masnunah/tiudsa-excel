# Project: Excel / unit-4

## What I did

* Downloaded a supermarket sales dataset from Kaggle
* Opened the CSV file in Excel and saved it as an `.xlsx` workbook
* Created a Revenue column using Unit Price × Quantity
* Converted the complete dataset into a proper Excel Table using Ctrl + T
* Sorted the Revenue column from highest to lowest
* Applied a filter to show only one product category
* Used a custom number filter to show transactions above a selected revenue amount

## Files / Links

* `Supermarket_Sales.xlsx`

## How to Run / Verify

1. Download and open `Supermarket_Sales.xlsx`
2. Click any cell inside the dataset
3. Check that the **Table Design** tab appears, confirming that the data is a proper Excel Table
4. Check the Revenue column to confirm that values are sorted from highest to lowest
5. Check the Category column to see the selected category filter
6. Check the Revenue column filter to confirm that a custom **Greater Than** number filter is applied

## Notes / References

* Dataset: Supermarket Sales Dataset from Kaggle

* Revenue was calculated using the formula:

  `Revenue = Unit Price × Quantity`

* The original data was downloaded as a CSV file and converted to an Excel `.xlsx` workbook

* The filtering and sorting settings are saved inside the workbook
